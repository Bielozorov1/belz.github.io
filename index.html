TrueFit Puzzle Cutter

TrueFit Puzzle Cutter is a Unity Editor tool that slices any image into perfectly matching jigsaw puzzle pieces. Tabs and holes share the same mathematical centers, giving clean seams and reliable piece snapping for 2D and UI-based puzzle games.

Core Features

Pixel-Perfect Puzzle Generation

Generate jigsaw pieces from any texture with exact tab and hole alignment, consistent padding, and transparent PNG output.

  • Grid-based slicing (rows × columns)
  • Configurable tab radius and center offset
  • Automatic safe padding calculation
  • Deterministic edge layout (stable between builds)
  • Editor-only workflow — no runtime overhead
For Developers

Designed for Production Use

Built for teams who care about visual quality and predictable results. Works great for mobile, casual, and educational projects.
